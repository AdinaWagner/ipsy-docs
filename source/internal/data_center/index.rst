* 1x `head node <medusa>`_ (medusa)

* 12x `compute nodes <compute_nodes>`_ (snake1-12)
* 1x `kinda-sorta-a-web server <mulder>`_ (mulder)
* 3x `network switches <network>`_ (IPMI, data, DMZ)
